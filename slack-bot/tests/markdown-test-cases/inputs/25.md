## Bolt for Python

**Bolt for Python** is a Python framework for building Slack apps quickly with the latest platform features.

### Key Features:

1. **Setup**: Requires Python 3.7+ and can be installed via `pip install slack_bolt`

2. **Two App Types**:
   - **Synchronous App**: Using `App()` constructor
   - **Async App**: Using `AsyncApp()` with asyncio/await pattern

3. **Running Modes**:
   - **HTTP Mode**: Uses ngrok for local development, listens on port 3000
   - **Socket Mode**: Direct WebSocket connection, no ngrok needed

4. **Event Listeners**: Support for multiple event types:
   - Actions (buttons, select menus, etc.)
   - Commands (slash commands)
   - Events (Events API)
   - Functions (workflow steps)
   - Messages (with pattern matching)
   - Options (external data sources)
   - Shortcuts (global or message)
   - Views (modal submissions)

5. **Listener Arguments**: Functions receive helpful arguments like:
   - `body`, `payload`, `context`
   - `ack` (acknowledge events)
   - `say` (send messages)
   - `client` (Web API client)
   - `respond` (respond to events)
   - `logger`, `complete`, `fail`

6. **Async Support**: Full async/await support with AIOHTTP and compatibility with frameworks like Sanic, FastAPI, and Starlette

The repository includes extensive examples and documentation at [docs.slack.dev/tools/bolt-python/](https://docs.slack.dev/tools/bolt-python/).
