## Bolt for Python

**What it is:** A Python framework to build Slack apps quickly using the latest platform features.

**Key Features:**

### Setup & Requirements
- Python 3.7+ required
- Install via: `pip install slack_bolt`

### Two App Types:
1. **Synchronous App** - Standard Python app using `App` constructor
2. **Asynchronous App** - Uses asyncio with `AsyncApp` constructor (requires aiohttp)

### Running Modes:
1. **HTTP Mode** - Standard web server requiring ngrok for development
2. **Socket Mode** - Uses WebSocket connection, no ngrok needed (requires `SocketModeHandler`)

### Core Functionality:
The framework provides listener methods for various Slack events:
- **Actions** - Block Kit interactions (buttons, menus, etc.)
- **Commands** - Slash commands
- **Events** - Events API events
- **Functions** - Custom workflow steps
- **Messages** - Convenience method for message events
- **Options** - External select menu data
- **Shortcuts** - Global or message shortcuts
- **Views** - Modal submissions

### Key Arguments in Listeners:
- `body` - Full request body
- `payload` - Event-specific data
- `ack` - Acknowledge receipt (required)
- `say` - Send message to channel
- `respond` - Respond to events with response_url
- `client` - Web API client
- `logger` - Logging instance
- `complete`/`fail` - For custom workflow steps

The repository includes extensive examples and adapters for various async web frameworks (Sanic, FastAPI, Starlette).
