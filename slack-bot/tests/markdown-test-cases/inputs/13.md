Here's a summary of Bolt for Python in table format:

## Bolt for Python Summary

### Basic Information
| **Aspect** | **Details** |
|------------|-------------|
| **Purpose** | Python framework to build Slack apps quickly |
| **Python Version** | 3.7+ required |
| **Installation** | `pip install slack_bolt` |
| **Async Support** | Yes, via `AsyncApp` with aiohttp |

### App Modes
| **Mode** | **Class** | **Use Case** | **Requirements** |
|----------|-----------|--------------|------------------|
| HTTP Mode | `App` | Standard web server | SLACK_SIGNING_SECRET, SLACK_BOT_TOKEN, ngrok |
| Socket Mode | `App` + `SocketModeHandler` | No public URL needed | SLACK_APP_TOKEN, SLACK_BOT_TOKEN |
| Async Mode | `AsyncApp` | Async/await pattern | aiohttp library |

### Event Listener Methods
| **Method** | **Purpose** | **Example Use Case** |
|------------|-------------|----------------------|
| `app.action(action_id)` | Block Kit element interactions | Button clicks, select menus, date pickers |
| `app.command(command_name)` | Slash commands | `/hello-bot` commands |
| `app.event(event_type)` | Events API events | Message posted, user joined |
| `app.message([pattern])` | Message events | Listen for specific message patterns |
| `app.shortcut(callback_id)` | Global/message shortcuts | Quick actions from Slack UI |
| `app.view(callback_id)` | Modal submissions | Form submissions |
| `app.function(callback_id)` | Custom workflow steps | Workflow automation |
| `app.options(action_id)` | External data sources | Dynamic select menu options |

### Listener Function Arguments
| **Argument** | **Type** | **Description** | **Available For** |
|--------------|----------|-----------------|-------------------|
| `body` | dict | Complete request body | All events |
| `payload` | dict | Core event data | All events |
| `context` | dict | Event context (botId, etc.) | All events |
| `ack` | function | Acknowledge event receipt (required) | Actions, shortcuts, views, commands, options |
| `say` | function | Send message to channel | Events with channel_id |
| `respond` | function | Respond to event | Events with response_url |
| `client` | Web API client | Make API calls | All events |
| `logger` | Logger | Python logging instance | All events |
| `complete` | function | Signal workflow step success | Custom workflow steps |
| `fail` | function | Signal workflow step failure | Custom workflow steps |

### Environment Variables
| **Variable** | **Format** | **Required For** | **Purpose** |
|--------------|-----------|------------------|-------------|
| SLACK_BOT_TOKEN | `xoxb-***` | All modes | Bot authentication |
| SLACK_SIGNING_SECRET | `***` | HTTP mode only | Request verification |
| SLACK_APP_TOKEN | `xapp-***` | Socket mode only | WebSocket connection |
