## Flask Adapter Overview

### üì¶ **Core Component: `SlackRequestHandler`**

The Flask adapter provides a simple handler class that bridges Flask and Bolt:

**Key Features:**
1. **Request/Response Translation**: Converts between Flask and Bolt formats
2. **OAuth Flow Support**: Handles installation and OAuth redirect routes automatically
3. **Event Dispatching**: Routes POST requests to the Bolt app

### üîß **How It Works:**

The `SlackRequestHandler` class:

```python
class SlackRequestHandler:
    def __init__(self, app: App):
        self.app = app
    
    def handle(self, req: Request) -> Response:
```

**Handles 3 types of requests:**

1. **GET `/install`** - OAuth installation flow (if OAuth is enabled)
2. **GET `/oauth_redirect`** - OAuth callback after authorization
3. **POST** - All Slack events, actions, commands, etc.

**Helper functions:**
- `to_bolt_request()` - Converts Flask Request ‚Üí BoltRequest
- `to_flask_response()` - Converts BoltResponse ‚Üí Flask Response

### üìù **Usage Example:**

**Basic App (app.py):**
```python
from slack_bolt import App
from slack_bolt.adapter.flask import SlackRequestHandler
from flask import Flask, request

# Create Bolt app
app = App()

@app.event("app_mention")
def event_test(body, say, logger):
    logger.info(body)
    say("What's up?")

# Create Flask app and handler
flask_app = Flask(__name__)
handler = SlackRequestHandler(app)

@flask_app.route("/slack/events", methods=["POST"])
def slack_events():
    return handler.handle(request)
```

**With OAuth (oauth_app.py):**
```python
# Same setup, but add OAuth routes:

@flask_app.route("/slack/install", methods=["GET"])
def install():
    return handler.handle(request)

@flask_app.route("/slack/oauth_redirect", methods=["GET"])
def oauth_redirect():
    return handler.handle(request)
```
