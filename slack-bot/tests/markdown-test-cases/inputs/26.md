## Bolt for Python - Slack App Framework

**bolt-python** is a Python framework for building Slack apps quickly using the latest platform features.

### Key Features:

1. **Easy Setup** - Requires Python 3.7+ and simple pip installation:
   ```bash
   pip install slack_bolt
   ```

2. **Two App Types**:
   - **Synchronous App** - Using the standard `App` class
   - **Async App** - Using `AsyncApp` with asyncio and AIOHTTP

3. **Running Options**:
   - **HTTP Mode** - Standard webhook-based app with ngrok for local development
   - **Socket Mode** - No need for public URLs or ngrok, uses WebSocket connections

4. **Event Listeners** - Methods to handle various Slack events:
   - `app.action()` - Block Kit interactions (buttons, menus, etc.)
   - `app.command()` - Slash commands
   - `app.event()` - Events API events
   - `app.message()` - Message events with pattern matching
   - `app.shortcut()` - Global/message shortcuts
   - `app.view()` - Modal submissions
   - `app.function()` - Custom workflow steps
   - `app.options()` - External select menu options

5. **Listener Arguments** - Functions receive helpful utilities:
   - `ack` - Acknowledge events
   - `say` - Send messages to channels
   - `respond` - Respond to interactive events
   - `client` - Web API client
   - `body`/`payload` - Event data
   - `logger` - Built-in logging

6. **Framework Adapters** - Built-in support for async frameworks like Sanic, FastAPI, and Starlette

The repository includes extensive examples and comprehensive documentation at https://docs.slack.dev/tools/bolt-python/
