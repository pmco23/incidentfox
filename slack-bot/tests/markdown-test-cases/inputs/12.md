## Bolt for Python - Overview

**Bolt for Python** is a framework from Slack to build Slack apps quickly using the latest platform features.

### Key Features:

1. **Quick Setup**: Requires Python 3.7+ and simple pip installation (`pip install slack_bolt`)

2. **Two App Types**:
   - **Synchronous App** (`App`) - Standard Python apps
   - **Asynchronous App** (`AsyncApp`) - Built with asyncio/AIOHTTP for async/await pattern

3. **Running Modes**:
   - **HTTP Mode**: Standard web server mode using ngrok for local development
   - **Socket Mode**: No ngrok needed, uses WebSocket connections with `SocketModeHandler`

4. **Event Listeners**: Methods to handle various Slack events:
   - `app.action()` - Block Kit element interactions
   - `app.command()` - Slash commands
   - `app.event()` - Events API events
   - `app.message()` - Message events
   - `app.shortcut()` - Shortcuts
   - `app.view()` - Modal submissions
   - `app.function()` - Custom workflow steps
   - `app.options()` - External select menu options

5. **Listener Arguments**: Functions receive helpful arguments like:
   - `body`, `payload` - Event data
   - `ack` - Acknowledge receipt
   - `say` - Send messages to channel
   - `respond` - Respond to events
   - `client` - Web API client
   - `logger` - Logging utility

6. **Framework Support**: Built-in adapters for async frameworks like Sanic, FastAPI, and Starlette

The repository includes documentation, examples, and comprehensive Python module references for building Slack apps.
