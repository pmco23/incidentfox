### Purpose:
The listener module handles the processing of incoming requests from Slack. Listeners:
- Match incoming requests against predefined conditions
- Acknowledge requests from Slack
- Process request data
- Send responses back to Slack

The module provides both **synchronous** (thread-based) and **asynchronous** (asyncio-based) implementations, with separate runners, error handlers, and lifecycle hooks for each execution model.
