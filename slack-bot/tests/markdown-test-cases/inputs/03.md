## Bolt for Python Overview

**Bolt for Python** is a framework for building Slack apps quickly using the latest platform features. It's the official Python framework from Slack.

### Key Features:

1. **Easy Setup**: Requires Python 3.7+ and can be installed via pip with `pip install slack_bolt`

2. **Two App Types**:
   - **Synchronous App**: Using the standard `App` class
   - **Async App**: Using `AsyncApp` with asyncio/aiohttp for asynchronous operations

3. **Multiple Running Modes**:
   - Standard HTTP mode (with ngrok for local development)
   - Socket Mode (no need for public URLs or ngrok)

4. **Event Listeners**: Built-in methods to handle various Slack events:
   - Actions (buttons, select menus, etc.)
   - Slash commands
   - Events API events
   - Shortcuts
   - View submissions (modals)
   - Custom workflow functions
   - Message events
   - Options requests

5. **Useful Arguments in Listeners**:
   - `ack` - Acknowledge events
   - `say` - Send messages to channels
   - `respond` - Respond to incoming events
   - `client` - Web API client
   - `logger` - Built-in logging
   - `payload`/`body` - Event data

6. **Framework Support**: Built-in adapters for async web frameworks like Sanic, FastAPI, and Starlette

The repository includes extensive examples and comprehensive documentation at https://docs.slack.dev/tools/bolt-python/
