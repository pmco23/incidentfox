## Bolt for Python - Simple Summary

| **Category** | **Details** |
|--------------|-------------|
| **What it is** | Python framework for building Slack apps |
| **Installation** | `pip install slack_bolt` |
| **Python Version** | 3.7+ |
| **Main Classes** | `App` (sync) or `AsyncApp` (async) |
| **Running Modes** | HTTP (needs ngrok) or Socket Mode (no ngrok) |
| **Listen to Events** | `.action()`, `.command()`, `.event()`, `.message()`, `.shortcut()`, `.view()` |
| **Key Functions** | `ack()` - acknowledge, `say()` - send message, `respond()` - reply |
| **Required Tokens** | `SLACK_BOT_TOKEN` (always), `SLACK_SIGNING_SECRET` (HTTP), `SLACK_APP_TOKEN` (Socket) |
| **Documentation** | https://docs.slack.dev/tools/bolt-python/ |
| **Examples** | https://github.com/slackapi/bolt-python/tree/main/examples |
