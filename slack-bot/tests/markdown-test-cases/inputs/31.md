## Bolt for Python - Overview

**Bolt for Python** is a framework for building Slack apps quickly using the latest platform features.

### Key Features:

1. **Setup Requirements:**
   - Python 3.7+ required
   - Install via `pip install slack_bolt`

2. **Two App Types:**
   - **Synchronous App** (`App`) - Standard blocking mode
   - **Async App** (`AsyncApp`) - Uses asyncio with aiohttp

3. **Running Modes:**
   - **HTTP Mode** - Traditional webhook-based with ngrok for development
   - **Socket Mode** - No need for public URLs or ngrok, uses WebSocket connection

4. **Event Listeners:**
   The framework provides methods to listen for various Slack events:
   - `.action()` - Block Kit element interactions
   - `.command()` - Slash commands
   - `.event()` - Events API events
   - `.function()` - Custom workflow steps
   - `.message()` - Message events with pattern matching
   - `.options()` - External data source requests
   - `.shortcut()` - Global or message shortcuts
   - `.view()` - Modal submissions

5. **Listener Function Arguments:**
   - `body` - Full request body
   - `payload` - Event-specific data
   - `context` - Event context and metadata
   - `ack` - Acknowledgment function (required)
   - `respond` - Quick response utility
   - `say` - Send messages to channel
   - `client` - Web API client
   - `logger` - Built-in logging
   - `complete`/`fail` - For custom workflow steps

6. **Web Framework Adapters:**
   Built-in adapters for various async frameworks like Sanic, FastAPI, and Starlette

The repository includes extensive examples and documentation at https://docs.slack.dev/tools/bolt-python/
