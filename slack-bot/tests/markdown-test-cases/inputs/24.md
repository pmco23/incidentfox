## Bolt for Python - Overview

**Bolt for Python** is a framework by Slack for building Slack apps quickly using the latest platform features.

### Key Features:

1. **Requirements**: Python 3.7+ required

2. **Installation**:
   ```bash
   pip install slack_bolt
   ```

3. **Two App Types**:
   - **Synchronous App** (`App`) - Standard Python
   - **Async App** (`AsyncApp`) - Uses asyncio and AIOHTTP

4. **Running Modes**:
   - **HTTP Mode**: Requires exposing an endpoint (e.g., via ngrok) and uses `SLACK_SIGNING_SECRET` and `SLACK_BOT_TOKEN`
   - **Socket Mode**: No public URL needed, uses `SocketModeHandler` with `SLACK_APP_TOKEN` and `SLACK_BOT_TOKEN`

5. **Event Listeners**: The framework provides decorators/methods for:
   - Actions (buttons, select menus, etc.)
   - Commands (slash commands)
   - Events (Events API)
   - Functions (custom workflow steps)
   - Messages (convenience method for message events)
   - Options (external data sources)
   - Shortcuts (global or message shortcuts)
   - Views (modal submissions)

6. **Listener Arguments**: Functions receive helpful arguments like:
   - `body`, `payload` - Event data
   - `ack` - Acknowledge the event
   - `say` - Send messages to channels
   - `respond` - Respond to events with response_url
   - `client` - Web API client
   - `logger` - Built-in logger
   - `complete`/`fail` - For custom workflow steps

7. **Async Support**: Full async/await support with `AsyncApp` and integration with frameworks like Sanic, FastAPI, and Starlette

The repository is well-documented and includes examples for various use cases!
