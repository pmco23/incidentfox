## Detailed Folder Exploration

### **1. Middleware (`/slack_bolt/middleware/`)**
The middleware system processes requests before they reach listeners:

**Built-in Middleware Classes:**
- `SslCheck` - SSL verification
- `RequestVerification` - Validates Slack request signatures
- `SingleTeamAuthorization` / `MultiTeamsAuthorization` - Auth handling
- `IgnoringSelfEvents` - Prevents bot from responding to itself
- `UrlVerification` - Handles Slack's URL verification challenge
- `AttachingFunctionToken` - Attaches function tokens
- `Assistant` - AI assistant middleware

**Middleware Types:**
- Sync: `middleware.py`, `custom_middleware.py`
- Async: `async_middleware.py`, `async_custom_middleware.py`
- Error handling: `middleware_error_handler.py`
