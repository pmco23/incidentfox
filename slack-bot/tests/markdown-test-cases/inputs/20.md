## FastAPI Adapter Overview

### üéØ **Key Point:** FastAPI uses the Starlette adapter under the hood
Since FastAPI is built on top of Starlette, the FastAPI adapter simply re-exports the Starlette handlers.

### üì¶ **Two Handler Options:**

1. **`SlackRequestHandler`** - For sync Bolt apps (uses regular `App`)
2. **`AsyncSlackRequestHandler`** - For async Bolt apps (uses `AsyncApp`)

### üîß **How It Works:**

**Core features:**
- **Request/Response Translation**: Converts Starlette Request ‚Üî BoltRequest/BoltResponse
- **OAuth Support**: Handles GET requests for `/install` and `/oauth_redirect`
- **Event Dispatching**: Routes POST requests to Bolt app
- **Cookie Support**: Properly handles cookies in responses
- **Custom Context Properties**: Can inject additional context data

### üìù **Usage Examples:**

#### **1. Sync App (app.py):**
```python
from slack_bolt import App
from slack_bolt.adapter.fastapi import SlackRequestHandler
from fastapi import FastAPI, Request

# Bolt app (sync)
app = App()
app_handler = SlackRequestHandler(app)

@app.event("app_mention")
def handle_app_mentions(body, say, logger):
    logger.info(body)
    say("What's up?")

# FastAPI app
api = FastAPI()

@api.post("/slack/events")
async def endpoint(req: Request):
    return await app_handler.handle(req)  # Handler is async even for sync Bolt app!
```

#### **2. Async App (async_app.py) - RECOMMENDED:**
```python
from slack_bolt.async_app import AsyncApp
from slack_bolt.adapter.fastapi.async_handler import AsyncSlackRequestHandler
from fastapi import FastAPI, Request

# Bolt app (async)
app = AsyncApp()
app_handler = AsyncSlackRequestHandler(app)

@app.event("app_mention")
async def handle_app_mentions(body, say, logger):
    logger.info(body)
    await say("What's up?")  # Use await with async app

# FastAPI app
api = FastAPI()

@api.post("/slack/events")
async def endpoint(req: Request):
    return await app_handler.handle(req)
```

#### **3. With OAuth (async_oauth_app.py):**
```python
api = FastAPI()

@api.post("/slack/events")
async def endpoint(req: Request):
    return await app_handler.handle(req)

@api.get("/slack/install")
async def install(req: Request):
    return await app_handler.handle(req)

@api.get("/slack/oauth_redirect")
async def oauth_redirect(req: Request):
    return await app_handler.handle(req)
```

#### **4. With Custom Context Properties (async_app_custom_props.py):**
```python
from fastapi import Depends

def get_foo():
    yield "FOO"

@api.post("/slack/events")
async def endpoint(req: Request, foo: str = Depends(get_foo)):
    # Pass custom properties to handler
    return await app_handler.handle(req, {"foo": foo})

@app.event("app_mention")
async def handle_app_mentions(context, say, logger):
    # Access custom property in context
    assert context.get("foo") == "FOO"
    await say("What's up?")
```

### ‚ú® **Key Advantages:**

1. **Async-First**: Perfect for high-performance async applications
2. **FastAPI Features**: Full access to dependency injection, validation, OpenAPI docs
3. **Custom Context**: Can inject custom properties via `addition_context_properties` parameter
4. **Type Safety**: Works great with FastAPI's type hints
5. **Modern**: Uses latest Python async/await patterns

### üìä **Comparison: Sync vs Async:**

| Feature | Sync (`SlackRequestHandler`) | Async (`AsyncSlackRequestHandler`) |
|---------|------------------------------|-------------------------------------|
| Bolt App Type | `App` | `AsyncApp` |
| Listener Functions | Regular functions | `async def` functions |
| Utility Methods | `say()`, `ack()` | `await say()`, `await ack()` |
| Performance | Good | Excellent (concurrent) |
| Use Case | Simple apps | High-traffic apps |

**Recommendation:** Use `AsyncSlackRequestHandler` with `AsyncApp` for best performance with FastAPI! üöÄ
